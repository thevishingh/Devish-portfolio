# 🚀 Vishal Singh - React Portfolio

**Advanced React portfolio built with Framer Motion, GSAP, Three.js showcasing 2 years React + 10 months web internship expertise.**

[]
[]

## 🎯 Features
- Multi-page React Router navigation (Home, About, Skills, Journey, Contact)
- 3D interactive elements with Three.js + React Three Fiber
- Advanced animations: Framer Motion + GSAP scroll triggers
- Custom animated cursor trail effect
- Futuristic neon design with dark/light mode toggle
- Smooth parallax scrolling & section reveals
- Fully responsive desktop → mobile
- Performance optimized with preload animations

## 🛠 Tech Stack
| Category | Technologies |
|----------|--------------|
| **Core** | React 18, React Router, Tailwind CSS |
| **Animations** | Framer Motion, GSAP, Lenis |
| **3D** | Three.js, @react-three/fiber, @react-three/drei |
| **Utils** | React Icons, Lucide React |

## 🚀 Quick Start
```bash
https://github.com/thevishingh/Devish-portfolio.git
cd Devish-portfolio
npm install
npm run dev
```

## 📁 Folder Structure
```
src/
├── components/
│   ├── common/     # Navbar, Footer, ThemeToggle
│   ├── ui/         # Button, Card, CursorTrail
│   └── sections/   # Hero, About, Skills, etc.
├── data/           # skills.json, journey.json
├── hooks/          # useMouse, useScrollAnimations
├── lib/            # GSAP utils, Three.js scenes
├── pages/          # Routed pages
└── utils/          # Animation helpers
```

## 🎨 Easy Customization
Update JSON files in `src/data/`:
```
skills.json      # Tech stack + proficiency
journey.json     # Timeline events
```
